========
Features
========

- View EXIF, IPTC, XMP, and video metadata
- Edit metadata values (Linux/macOS fully supported)
- Open files from disk or direct URL
- Professional HTML report generator
- Export metadata to CSV or JSON
- Context menu with delete + add row options
- Clean GUI using ttkbootstrap
- Cross-platform interface

Planned features:

- Batch editing
- CLI mode
- Plugin system
