ExifPlus
========

A simple EXIF/XMP/IPTC metadata viewer and editor.

Quick start
-----------

Install (recommended):

.. code-block:: bash

    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    pip install -e .

Run
---

.. code-block:: bash

    python -m exifplus
