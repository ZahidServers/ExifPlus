-----------------
What is ExifPlus?
-----------------

ExifPlus is a cross-platform tool designed to make metadata analysis simple and
powerful. It allows you to:

- View EXIF, IPTC, XMP, and video metadata  
- Edit metadata (Linux/macOS fully supported; Windows has limitations)
- Open files from disk **or directly from a URL**
- Export metadata to **CSV or JSON**
- Generate **professional HTML reports** for Bugcrowd / HackerOne submissions
- Use a clean, modern GUI built with *ttkbootstrap*
